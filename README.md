## About

Adds support for ADF/HDF files to DirectoryOpus.

This fork updates the `adflib` dependency and modernizes the build structure, detaching the project from Visual Studio.

- To use this plugin you need a Directory Opus (12 or above).
- To install the plugin - simply copy relevant binaries to `C:\Program Files\GPSoftware\Directory Opus\VFSPlugins`.
